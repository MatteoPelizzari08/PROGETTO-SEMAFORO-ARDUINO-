# PROGETTO-SEMAFORO-ARDUINO-
in questo progetto utilizzo Arduino per fare la simulazione del semaforo e, ogni volta che premo il bottone, salva lo stato in un file data.csv 

il file denominato "sketch_feb6" è il file per aprire in Arduino IDE

il file "main.py" contiene il codice per salvare lo stato del bottone --> bisogna aprirlo in Visual Studio Code

il filde "dati.csv" contiene quando il bottone è stato premuto --> aprirlo in Visual Studio Code
